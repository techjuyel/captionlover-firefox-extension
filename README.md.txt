A minimal Firefox extension that works like a smart bookmark for https://captionlover.com

## Features
- One click opens CaptionLover
- Lightweight
- No data collection
- Firefox Manifest V3 compatible

## Installation (Temporary)

1. Open Firefox
2. Go to: about:debugging
3. Click "This Firefox"
4. Click "Load Temporary Add-on"
5. Select manifest.json

## Build for Store

Zip these files directly (no folder):

manifest.json  
background.js  
icon.png
