# Caption Lover - Browser Extension

**Caption Lover** হলো একটি হালকা ওজনের ব্রাউজার এক্সটেনশন যা আপনাকে সরাসরি আপনার ব্রাউজার থেকে সেরা সোশ্যাল মিডিয়া ক্যাপশন এবং স্ট্যাটাস খুঁজে পেতে সাহায্য করে। এটি মূলত **[CaptionLover.com](https://captionlover.com)** এর অফিসিয়াল এক্সটেনশন।

## প্রধান বৈশিষ্ট্যসমূহ (Features)
- **Instant Access:** এক ক্লিকেই হাজার হাজার ট্রেন্ডি ক্যাপশন এবং স্ট্যাটাস।
- **Categorized Content:** রোমান্টিক, মোটিভেশনাল, দুঃখের বা মজার—সব ধরণের স্ট্যাটাস আলাদা ক্যাটাগরিতে।
- **Lightweight:** ব্রাউজারের পারফরম্যান্সে কোনো প্রভাব ফেলবে না।
- **Privacy Focused:** আপনার কোনো ব্যক্তিগত তথ্য সংগ্রহ করা হয় না।

## ইন্সটলেশন (Installation)
বর্তমানে আপনি এটি ম্যানুয়ালি লোড করতে পারেন:
1. এই রিপোজিটরি থেকে জিপ ফাইলটি ডাউনলোড করুন।
2. ফাইলটি আনজিপ করুন।
3. আপনার ব্রাউজারের (Firefox/Chrome) এক্সটেনশন ম্যানেজমেন্ট পেজে যান।
4. `Developer Mode` অন করুন এবং `Load unpacked` এ ক্লিক করে ফোল্ডারটি সিলেক্ট করুন।
